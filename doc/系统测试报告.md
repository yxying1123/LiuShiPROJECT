### 3.2 数据结构分析

以 `export_D28_IT1_T8.csv` 为例：

**列结构**：
| 列名 | 类型 | 说明 |
|-----|------|-----|
| FSC-A | 数值 | 前向散射面积（细胞大小） |
| FSC-H | 数值 | 前向散射高度 |
| FSC-W | 数值 | 前向散射宽度 |
| SSC-A | 数值 | 侧向散射面积（细胞颗粒度） |
| KLRG1 | 数值 | 杀伤细胞凝集素样受体 |
| LY6C | 数值 | 淋巴细胞抗原6C |
| 62l | 数值 | L-选择素 |
| CD44 | 数值 | 白细胞分化抗原44 |
| CD3 | 数值 | T细胞受体复合物 |
| CD103 | 数值 | 整合素αE |
| BV480-A | 数值 | 亮度通道 |
| CD8 | 数值 | 细胞毒性T细胞标志 |
| CD4 | 数值 | 辅助T细胞标志 |
| PD1 | 数值 | 程序性死亡受体1 |
| CD127 | 数值 | IL-7受体α链 |
| Time | 数值 | 采集时间 |

**数据特点**：
- 标准流式细胞术数据格式
- 包含细胞表面标志物表达量
- 适用于免疫细胞分群分析
