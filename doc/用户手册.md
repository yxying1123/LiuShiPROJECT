## 系统概述

### 1.1 产品简介

单细胞数据分析系统是一款面向流式细胞术（Flow Cytometry）和质谱流式（CyTOF）单细胞数据的专业分析平台。系统提供从数据上传、降维可视化、聚类分析到热图展示的一站式解决方案，帮助研究人员快速发现数据中的生物学模式和细胞亚群。

### 1.2 核心功能

| 功能模块 | 功能描述 |
|---------|---------|
| **数据管理** | 支持 CSV、FCS 等多种格式文件的上传、存储和管理 |
| **降维可视化** | 基于 UMAP 算法的高维数据降维和散点图展示 |
| **聚类分析** | 基于 Phenograph 算法的细胞聚类，自动识别细胞亚群 |
| **热图分析** | 层次聚类热图展示，支持行/列聚类树可视化 |
| **数据导出** | 支持分析结果的图片、CSV 数据导出 |

### 1.3 适用场景

- 免疫细胞分群分析
- 肿瘤微环境细胞异质性研究
- 药物处理后细胞状态变化分析
- 单细胞水平的生物标志物发现

## 系统架构

### 2.1 整体架构

系统采用前后端分离架构：

```
┌─────────────────────────────────────────────────────────────┐
│                        前端 (cluster)                        │
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────────────┐ │
│  │  文件管理    │  │  散点图分析  │  │    热图聚类分析      │ │
│  │  FileList   │  │  Scatter    │  │    Heatmap          │ │
│  └─────────────┘  └─────────────┘  └─────────────────────┘ │
│                         React + Vite                        │
└─────────────────────────────────────────────────────────────┘
                              │
                              │ HTTP/REST API
                              ▼
┌─────────────────────────────────────────────────────────────┐
│                       后端 (cluster-End)                     │
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────────────┐ │
│  │   文件服务   │  │  UMAP降维   │  │   Phenograph聚类    │ │
│  │   FastAPI   │  │   算法      │  │      算法          │ │
│  └─────────────┘  └─────────────┘  └─────────────────────┘ │
│                      Python + FastAPI                       │
└─────────────────────────────────────────────────────────────┘
```

### 2.2 技术栈

**前端技术栈：**
- **框架**: React 18 + Vite
- **UI组件**: Radix UI + Tailwind CSS
- **可视化**: Plotly.js + D3.js
- **状态管理**: React Context
- **HTTP客户端**: Axios

**后端技术栈：**
- **框架**: FastAPI (Python)
- **数据处理**: Pandas, NumPy
- **降维算法**: UMAP-learn
- **聚类算法**: Phenograph
- **统计分析**: SciPy, Scikit-learn

## 功能模块详解

### 5.1 文件管理模块

#### 5.1.1 功能概述

文件管理模块提供数据文件的上传、查看、删除和下载功能。支持批量上传多个文件，自动识别文件格式和列信息。

#### 5.1.2 支持的文件格式

| 格式 | 扩展名 | 说明 |
|-----|-------|-----|
| CSV | .csv | 逗号分隔值文件，最常用格式 |
| 文本 | .txt | 制表符或空格分隔的数据文件 |
| FCS | .fcs | 流式细胞术标准数据格式 |

#### 5.1.3 文件上传流程

1. 点击"上传文件"按钮或拖拽文件到上传区域
2. 系统自动验证文件格式
3. 上传完成后显示文件列表
4. 点击文件可查看列信息和数值列

#### 5.1.4 API接口

| 接口 | 方法 | 说明 |
|-----|------|-----|
| `/files` | GET | 获取文件列表 |
| `/files/upload` | POST | 上传文件 |
| `/files/{filename}` | DELETE | 删除文件 |
| `/files/download/{filename}` | GET | 下载文件 |
| `/files/metadata` | POST | 获取文件元数据 |

### 5.2 散点图分析模块

#### 5.2.1 功能概述

散点图分析模块是系统的核心功能之一，提供高维单细胞数据的 UMAP 降维可视化。支持多种数据选择模式、降维重计算和交互式框选。

#### 5.2.2 工作流程

```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│  选择文件    │ -> │  选择变量   │ -> │  生成散点图  │
└─────────────┘    └─────────────┘    └─────────────┘
                                              │
                                              v
┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│  聚类分析    │ <- │  框选数据   │ <- │  交互选择   │
└─────────────┘    └─────────────┘    └─────────────┘
```

#### 5.2.3 操作模式

**模式一：原始变量散点图**
- 直接选择 X 轴和 Y 轴变量
- 适用于已降维的数据或特定参数分析
- 不同文件来源使用不同颜色标识

**模式二：UMAP 降维散点图**
- 选择多个维度进行 UMAP 降维
- 自动计算二维坐标并展示
- 支持重新选择数据子集再次降维

#### 5.2.4 交互功能

| 功能 | 操作方式 | 说明 |
|-----|---------|-----|
| 框选 | 鼠标拖拽 | 选择感兴趣的数据区域 |
| 缩放 | 滚轮/手势 | 放大/缩小查看细节 |
| 平移 | 拖拽画布 | 移动视图位置 |
| 悬停 | 鼠标悬停 | 显示数据点详细信息 |
| 图例筛选 | 点击图例 | 显示/隐藏特定来源数据 |

#### 5.2.5 降维参数

UMAP 降维使用的默认参数：

| 参数 | 默认值 | 说明 |
|-----|-------|-----|
| n_neighbors | 15 | 局部邻域大小，影响局部/全局结构平衡 |
| min_dist | 0.001 | 嵌入空间中点的最小距离 |
| target_weight | 0.5 | 目标权重 |
| metric | euclidean | 距离度量方式 |

### 5.3 聚类分析模块

#### 5.3.1 功能概述

基于 Phenograph 算法对选中的数据点进行聚类分析，自动识别细胞亚群。聚类结果可用于后续的热图分析和 ROE（Relative Observed vs Expected）分析。

#### 5.3.2 Phenograph 算法参数

| 参数 | 默认值 | 说明 |
|-----|-------|-----|
| k | 50 | k-近邻图中的邻居数 |
| resolution | 1.0 | 社区检测分辨率 |
| n_iterations | 20 | 迭代次数 |

#### 5.3.3 聚类流程

1. 在散点图中框选感兴趣的数据区域
2. 点击"聚类分析"按钮
3. 系统调用后端进行 Phenograph 聚类
4. 聚类结果以不同颜色显示在散点图上
5. 同时生成热图聚类树

### 5.4 热图聚类模块

#### 5.4.1 功能概述

热图聚类模块展示聚类结果的表达矩阵，支持行和列的层次聚类，帮助识别基因/蛋白表达模式和样本间相似性。

#### 5.4.2 视图选项

| 视图 | 说明 |
|-----|-----|
| 聚类热图树 | 显示带层次聚类树的热图 |
| 散点图 | 显示聚类后的散点图分布 |
| 聚类表 | 显示聚类均值矩阵数据表 |
| 散点表 | 显示原始散点数据表 |

#### 5.4.3 热图参数配置

| 参数 | 选项 | 说明 |
|-----|-----|-----|
| scale | column/row/none | 标准化方式 |
| rowMetric | euclidean/correlation | 行距离度量 |
| colMetric | euclidean/correlation | 列距离度量 |
| linkageMethod | complete/average/single | 层次聚类连接方式 |

#### 5.4.4 层次聚类树

系统使用 SciPy 计算层次聚类树，输出 D3.js 兼容的树形结构：

```json
{
  "name": "root",
  "children": [
    {
      "name": "cluster1",
      "distance": 0.3,
      "children": [
        { "name": "geneA" },
        { "name": "geneB" }
      ]
    },
    { "name": "geneC" }
  ]
}
```

### 5.5 数据导出模块

#### 5.5.1 导出选项

| 导出内容 | 格式 | 说明 |
|---------|-----|-----|
| 散点图 | PNG | 高分辨率散点图图片 |
| 热图 | PNG | 热图聚类树图片 |
| 散点数据 | CSV | 完整散点数据表 |
| 散点数据（精简） | CSV | 去除坐标和聚类列的数据 |
| 聚类表 | CSV | 聚类均值矩阵 |

#### 5.5.2 保存聚类结果

分析完成后，可以保存聚类结果以便后续查看：

1. 点击"保存结果"按钮
2. 输入保存名称
3. 结果保存到本地存储
4. 可在历史记录中查看

## 操作指南

### 6.1 快速入门

#### 步骤1：启动系统

```bash
# 终端1：启动后端
cd cluster-End
uvicorn main:app --reload --host 0.0.0.0 --port 8000

# 终端2：启动前端
cd cluster
npm run dev
```

#### 步骤2：上传数据

1. 打开浏览器访问 `http://localhost:5173`
2. 进入文件管理页面
3. 点击"上传文件"按钮
4. 选择 CSV 或 FCS 格式的数据文件
5. 等待上传完成

#### 步骤3：生成散点图

1. 进入散点图分析页面
2. 选择要分析的文件
3. 选择 X 轴和 Y 轴变量（或选择多维度进行 UMAP 降维）
4. 点击"生成散点图"

#### 步骤4：聚类分析

1. 在散点图中用鼠标框选感兴趣的区域
2. 点击"聚类分析"按钮
3. 等待聚类完成
4. 查看聚类热图树结果

### 6.2 详细操作流程

#### 6.2.1 文件上传详细步骤

1. **进入文件列表页面**
   - 点击左侧导航栏"文件列表"
   - 查看已上传的文件

2. **上传新文件**
   - 点击"上传文件"按钮
   - 在弹出的对话框中选择文件
   - 支持多选批量上传
   - 系统自动检测文件格式

3. **查看文件信息**
   - 点击文件名查看详细信息
   - 查看所有列名和数值列
   - 确认数据格式正确

4. **删除文件**
   - 点击文件卡片上的删除按钮
   - 确认删除操作

#### 6.2.2 散点图分析详细步骤

1. **选择分析模式**
   - **2D模式**：直接选择 X、Y 轴变量
   - **UMAP模式**：选择多个维度进行降维

2. **配置参数**
   - 选择要分析的文件
   - 选择变量（X/Y轴或多维度）
   - 设置采样行数（可选）
   - 选择要排除的列

3. **生成散点图**
   - 点击"生成散点图"按钮
   - 等待数据处理完成
   - 查看生成的散点图

4. **交互操作**
   - **框选**：按住鼠标左键拖拽选择区域
   - **缩放**：使用鼠标滚轮
   - **平移**：按住鼠标右键拖拽
   - **查看详情**：鼠标悬停在数据点上

5. **数据筛选**
   - 使用图例显示/隐藏特定来源
   - 使用颜色模式切换（数值/来源/聚类）

#### 6.2.3 聚类分析详细步骤

1. **选择数据**
   - 在散点图中框选要分析的数据点
   - 可以多次框选累积选择

2. **执行聚类**
   - 点击"聚类分析"按钮
   - 系统调用后端进行 Phenograph 聚类
   - 等待分析完成

3. **查看结果**
   - 散点图上显示聚类颜色
   - 进入热图页面查看聚类热图树
   - 查看聚类表和散点表

4. **保存结果**
   - 点击"保存结果"按钮
   - 输入结果名称
   - 保存到历史记录

#### 6.2.4 热图分析详细步骤

1. **查看聚类热图树**
   - 切换到"聚类热图树"标签
   - 查看热图和层次聚类树
   - 观察表达模式和聚类关系

2. **查看聚类表**
   - 切换到"聚类表"标签
   - 查看各聚类的均值矩阵
   - 可导出为 CSV

3. **查看散点表**
   - 切换到"散点表"标签
   - 查看原始数据表
   - 可导出为 CSV

4. **导出结果**
   - 点击"下载热图图片"
   - 或点击"下载聚类表"

### 6.3 典型使用场景

#### 场景1：基础降维可视化

## 数据格式说明

### 7.1 CSV 文件格式

#### 7.1.1 标准格式

```csv
Time,Event_length,CD3,CD4,CD8,Sample
1,12,150.5,200.3,50.2,Sample1
2,15,180.2,220.1,45.8,Sample1
3,11,120.8,180.5,60.3,Sample2
...
```

#### 7.1.2 格式要求

- 第一行为列名（header）
- 逗号分隔
- 数值列应为数字格式
- 支持缺失值（留空或 NA）

#### 7.1.3 推荐列名规范

| 列类型 | 示例 | 说明 |
|-------|-----|-----|
| 标记物 | CD3, CD4, CD8, CD45 | 流式抗体标记 |
| 样本信息 | Sample, Group, Batch | 样本分组信息 |
| 技术参数 | Time, Event_length | 仪器参数 |

### 7.2 FCS 文件格式

系统支持标准 FCS 3.0/3.1 格式文件，会自动解析：
- 参数名称（PnN）
- 参数标签（PnS）
- 补偿矩阵（如存在）

### 7.3 数据预处理

#### 7.3.1 自动处理

系统会自动进行以下预处理：
- 数值类型转换
- 缺失值处理
- 标准化（可选）

#### 7.3.2 采样策略

对于大数据集，系统支持随机采样：
- 设置每文件采样行数
- 使用随机种子保证可重复性
- 默认采样 1000 行

## 常见问题与解决方案

### 8.1 安装问题

#### Q1: Node.js 版本不兼容

**问题**：运行 `npm install` 时报版本错误

**解决**：
```bash
# 使用 nvm 切换到正确版本
nvm install 18
nvm use 18
```

#### Q2: Python 依赖安装失败

**问题**：安装 `phenograph` 或 `umap-learn` 失败

**解决**：
```bash
# 升级 pip
pip install --upgrade pip

# 先安装 numpy 和 Cython
pip install numpy Cython

# 再安装其他依赖
pip install -r requirements.txt
```

### 8.2 运行问题

#### Q3: 前端无法连接后端

**问题**：页面显示"连接失败"或数据加载失败

**解决**：
1. 检查后端服务是否启动：`curl http://localhost:8000/`
2. 检查前端 API 配置是否正确
3. 检查防火墙设置
4. 确认端口号未被占用

#### Q4: UMAP 降维速度慢

**问题**：大数据集降维耗时过长

**解决**：
1. 减少采样行数
2. 启用低内存模式（修改 `low_memory=True`）
3. 使用更高配置的机器
4. 减少降维维度数

#### Q5: 聚类分析失败

**问题**：点击聚类后无响应或报错

**解决**：
1. 确保已框选数据点
2. 检查框选区域是否包含足够数据（至少50个细胞）
3. 查看浏览器控制台错误信息
4. 检查后端日志

### 8.3 数据问题

#### Q6: 上传文件失败

**问题**：文件上传时报错

**解决**：
1. 检查文件格式是否为 CSV/TXT/FCS
2. 检查文件大小（建议 < 500MB）
3. 检查文件编码（推荐 UTF-8）
4. 确认文件未被其他程序占用

#### Q7: 散点图显示异常

**问题**：散点图显示为一条线或一个点

**解决**：
1. 检查选择的 X/Y 列是否为数值类型
2. 检查数据是否存在异常值
3. 尝试更换降维参数
4. 检查是否选择了正确的列

### 8.4 浏览器兼容性问题

#### Q8: 页面显示异常

**问题**：界面布局错乱或功能不可用

**解决**：
1. 使用推荐的浏览器（Chrome 90+、Firefox 90+）
2. 清除浏览器缓存
3. 禁用浏览器插件
4. 检查浏览器控制台错误

### 附录B：算法说明

#### B.1 UMAP 算法

UMAP（Uniform Manifold Approximation and Projection）是一种非线性降维技术，特别适用于高维单细胞数据的可视化。

**原理简述**：
1. 构建高维空间的模糊拓扑表示
2. 优化低维嵌入以保持拓扑结构
3. 使用交叉熵作为损失函数

**参数调优建议**：
- `n_neighbors`：小值（5-15）保留更多局部结构，大值（50-100）保留更多全局结构
- `min_dist`：小值使点更聚集，大值使点更分散

#### B.2 Phenograph 算法

Phenograph 是一种基于图的聚类算法，广泛用于单细胞数据分析。

**算法流程**：
1. 构建 k-近邻图
2. 计算 Jaccard 相似度
3. 使用 Louvain 算法进行社区检测

**参数说明**：
- `k`：邻居数，影响聚类粒度（小k产生更多小聚类）
- `resolution`：分辨率参数，影响聚类数量
